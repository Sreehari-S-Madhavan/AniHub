# AniHub

## Project Overview

AniHub is a web application for anime, manga, and light novel fans. It helps users discover new content, track what they are watching or reading, and join community discussions. The app focuses on legal and community-driven features.

## Why This Project Exists

This project is built by beginners for beginners. It follows these core principles:

- No piracy or illegal content
- No video hosting or streaming
- No AI-powered recommendations
- Focus on learning through step-by-step development
- Emphasis on community and structured discussions

The goal is to create a simple, useful tool while teaching web development skills.

## Tech Stack

The project uses these technologies:

Frontend:
- React
- Vite
- JavaScript
- CSS

Backend:
- Node.js
- Express.js
- REST API

Database:
- PostgreSQL

## Features (MVP)

Current features include:

- User authentication (register and login)
- Browse anime and manga lists
- View detailed anime pages
- Personal tracking for watching, completed, or on-hold items
- Discover content by genre using simple rules
- Community discussions with agree/disagree options
- Information on legal places to watch content
